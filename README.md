# rest-board
RESTful Board with Express

# How to use
1. install mysql.
2. input 'db_table_create_query.txt' in MySQL. (to create tables.)
3. correct '/routes/conf/db-con.js' file to suit your running env. (usersname, password, ..)
4. npm install
5. npm start

#Feature
1. RESTful
2. SPA(Single Page Application)
3. PJAX
4. Node.js(Express)
